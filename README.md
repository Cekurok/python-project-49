## The package contains five simple mathematics quiz games:

* Even check brain-even
* Calculator brain-calc
* Greatest common divider brain-gcd
* Arithmetic progression brain-progression
* Prime number brain-prime

## Installation
```bash
poetry install
```
## How it works
